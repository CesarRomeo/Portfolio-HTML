## ðŸ“¦ Installation & Usage

### Simple Setup

1. **Clone or download the repository**
   ```bash
   cd portfolio-cesar-html
   ```

2. **Open in browser**
   - Simply double-click `index.html` to open in your default browser
   - Or right-click and choose "Open with" your preferred browser

### Using a Local Server (Recommended)

For the best experience and to test all features:

**Option 1: Using Python**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Option 2: Using Node.js**
```bash
# Install http-server globally
npm install -g http-server

# Run server
http-server
```

**Option 3: Using VS Code**
- Install "Live Server" extension
- Right-click on `index.html`
- Select "Open with Live Server"

Then open your browser at `http://localhost:8000` (or the port shown)

## ðŸŽ¨ Customization

### Update Personal Information

Edit `index.html` to change:
- Personal details (name, email, phone, location)
- Work experience
- Education history
- Skills and technologies
- Social media links

### Modify Colors

Edit CSS variables in `styles.css`:

```css
:root {
  --primary: #667eea;        /* Main brand color */
  --secondary: #764ba2;      /* Secondary brand color */
  --bg-dark: #0f172a;        /* Dark background */
  --bg-medium: #1e293b;      /* Medium background */
  --text-light: #94a3b8;     /* Light text color */
}
```

### Adjust Layout

All responsive breakpoints are defined in `styles.css`:
- Desktop: > 968px
- Tablet: 768px - 968px
- Mobile: < 768px
